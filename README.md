## 👨‍👨‍👧 냉장고를 부탁해

  - 삼시세끼를 뭐 해먹을지 항상 고민인 1인 가구를 위한 레시피 추천 모바일 웹 서비스
 
<br>
 
 ## 🤷‍♂️ 사용자
 
  - 냉장고 속 식재료 정보를 기반으로 레시피를 추천 받고 싶거나 식재료를 관리나 소진을 효율적으로 하고싶은 사용자

<br>
